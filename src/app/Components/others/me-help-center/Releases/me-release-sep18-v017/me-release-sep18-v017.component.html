September 2018 (version 0.17) - Release Summary

September 2018 release of (version 0.17) - There are significant updates in this version, some of the key highlights include:

  • Reports

    • Report UI : Report UI has been revamped for ease of usage and better clarity.

    • Column Grouping : Columns grouping in the report provide better clarity on data grouping.

    • Column Resize : Allows user to expand the report column to visualize data clearly.

    • Row Expander : Provides user to with additional information within visible viewport.

  • Dashboard

    • Updates Widget : Update widget added to dashboard. This widget provides information on notifications, upgrades, announcements and new features.

    • Agentwise Dasboard : Agentwise summary has been removed from dashboard and is only available in collection report.

Release Details
Report UI

Report UI has been revamped for ease of usage and better clarity. Report filter has been conveniently placed on the left hand-side of the report. Report now has column resize and row expander for better data clarity.
